## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Requirements
- Docker with Docker Compose
- NodeJS (min v20)
- Yarn Packatemanger

## Installation

```bash
$ yarn install
```

## Start database

```bash
# Start the docker containers
docker compose up -d

```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
